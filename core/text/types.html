Note

The optional identifier names for parameters in a function type only have documentation purpose. They cannot be referenced from anywhere.

\[\begin{split}\begin{array}{llclll} \def\mathdef4081#1{{}}\mathdef4081{recursive type} & \href{../text/types.html#text-typedef}{\mathtt{typedef}} &\equiv& \def\mathdef4273#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4273{(}~~\def\mathdef4274#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4274{rec}~~\href{../text/types.html#text-typedef}{\mathtt{typedef}}~~\def\mathdef4275#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4275{)} \\ \end{array}\end{split}\]

Similarly, final sub types with no super-types can omit the \(\href{../syntax/types.html#syntax-subtype}{\mathsf{sub}}\) keyword and arguments:

\[\begin{split}\begin{array}{llclll} \def\mathdef4081#1{{}}\mathdef4081{sub type} & \href{../text/types.html#text-comptype}{\mathtt{comptype}} &\equiv& \def\mathdef4276#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4276{(}~~\def\mathdef4277#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4277{sub}~~\def\mathdef4278#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4278{final}~~\epsilon~~\href{../text/types.html#text-comptype}{\mathtt{comptype}}~~\def\mathdef4279#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef4279{)} \\ \end{array}\end{split}\]
